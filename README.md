# Smart Chef!

This site takes in 4 or more ingredients or your choosing, and uses it to create recipes generated by Hugging Face AI! Use it in case you ever need a recipe idea!

Uses serverless architecture to store the api used safely!

Live Demo: https://thesmartchef.netlify.app/

# Tech Stack
Frontend: React, React Markdown
AI: Hugging Face
Backend: Node.js (Netlify)
